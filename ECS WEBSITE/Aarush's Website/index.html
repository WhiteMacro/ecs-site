Our Journey

1977

Oracle Corporation was founded by Larry Ellison, Bob Miner, and Ed Oates, initially as Software Development Laboratories.

1983

Renamed to Oracle Corporation, becoming the first company to offer a commercial SQL relational database management system.

1995

Launched Oracle 7, introducing PL/SQL and distributed database capabilities, revolutionizing enterprise computing.

2010

Acquired Sun Microsystems, expanding into hardware and Java technology, strengthening our enterprise solutions.

2020

Launched Oracle Cloud Infrastructure (OCI), providing next-generation cloud services with superior performance and security.
